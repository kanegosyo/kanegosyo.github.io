What is KaNegosyo App?

KaNegosyo is a fully offline delivery and small business finance tracking system. It helps you monitor customers, track receivables, manage capital, calculate profit automatically, and maintain transaction history — all stored locally on your device.

Delivery & Receivable Tracking
  • Add customer name, date, amount, description & area
  • Automatic status detection (For Delivery / Pending / Overdue)
  • Mark items as Delivered with confirmation
  • Delivered history (max 50 stored)
  • Search by name, date, or area
  • Optional sort by area
  • Expandable item details
Capital Management
  • Add or deduct capital anytime
  • Automatic receivable total calculation
  • Real-time delivery summary counts
  • Total capital & receivable dashboard
Finance Panel (GCash & Cash on Hand)
  • Track GCash balance
  • Track Cash-on-Hand balance
  • Set base capital
  • Automatic profit calculation
  • Expandable finance panel
Cash-In / Cash-Out with Interest
  • Custom interest percentage
  • Auto interest computation
  • Minimum ₱5 interest for small amounts
  • Automatic balance adjustments
  • Transaction history (last 100 records)
Smart Input System
  • Name auto-suggestions
  • Area auto-suggestions
  • Auto-save frequent customers
  • Quick entry with today’s date pre-filled
Encrypted Backup & Restore
  • Password-protected backup file
  • Full localStorage export
  • Secure restore with password verification
  • 100% offline — no server involved
Offline Locked PWA
  • Network requests disabled
  • Fully functional offline
  • Installable on mobile & desktop
  • Self contained app it cannot receive any updates
